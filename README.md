## 🎨 About

**M3 Play** is a modern Android music player built with  
✨ **Material 3 Expressive UI**

It delivers a smooth and clean experience with powerful features like online streaming, lyrics support, and downloads.

---

## ✨ Features

- 🌐 Online music streaming  
- ⬇️ Download songs for offline listening  
- 🎨 Material 3 Expressive UI  
- ⚡ Smooth & fluid animations  
- 🎶 Lyrics support (auto scroll)  
- 🎛️ Custom player design  
- 🌗 Light & Dark mode  
- 💾 Lightweight & fast
